# Luxuper

**Luxury Computer Peripherals**
Web Frameworks Capstone Project

-----------------------------------------------------------------------------

## Overview

Luxuper is a professional luxury computer peripheral website,
focusing on selling high-end products such as mice, keyboards, headphones,
web-cameras, speakers, and more. The intent of this project is to demonstrate
a strong understanding and implementation of full-stack web development.

 - Server-side rendering
 - Express.js
 - Model View Controller architecture (MVC)
 - MySQL database integration
 - REST-style API endpoints

This project will serve as an example for a professional portfolio of a well
structured and maintained web application.
